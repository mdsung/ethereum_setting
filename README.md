# Ethereum Setting(Python 3.6.8)
> Set the private network with python

In order to simulate Ethereum data transaction, set the private ethereum network with python

## Prerequisite
    * python 3.6
    * go 1.14
    * GETH 1.9

## Installation

OS X & Linux:

```sh
pip install -r requirements.txt
```
